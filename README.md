# PHP-SQL
Small library to simplify SQL queries and add encryption features.

## Setup
* Create a configuration file named `config.json` with content:
```json
{
    "hostname": "localhost",
    "database": "",
    "username": "",
    "password": "",
    "encrypt-key": "",
    "encrypt-second-key": ""
}
```
